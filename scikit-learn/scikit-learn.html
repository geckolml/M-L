Notas de Scikit-Learn¶

Machine Learning(Aprendizaje Automtico), es una rama de la inteligencia artificial que estudia el diseo de algoritmos que puedan aprender.

Tpicas tareas del machine learning son el aprendizaje de conceptos,el desarrollo de modelos de prediccin, la bsqueda de patrones predictivos y el clustering. Estas tareas son aprendidas a travs de los datos disponibles observados a travs de experiencias o instrucciones. Podemos dividir los algoritmos del machine learning en 3 categorias:

  • Aprendizaje supervisado
  • Aprendizaje no supervisado
  • Aprendizaje por refuerzo

scikit es un conjunto de paquetes en Python, para temas especficos:

  • scikit-image: Contiene rutina para procesamiento de imgenes en Scipy.
  • scikit-monaco: Paquete relacionado a la integracin de Montecarlo.
  • timeseries: Un paquete para la manipulacin de series de tiempo.
  • scikit-learn: Un conjunto de mdulos en Python, para Machine Learning y Data Minning.

La librera scikit-learn proporciona una gran nmero de herramientas para la ciencia de datos y data minning en Python centrandose en el machine learning:

  • scikit-learn posee excelente documentacin como el tutorial inicial o los ejemplos.
  • Posee muchos de lo algoritmos del machine learning.
  • Todo se trabaja sobre github.
  • Cada mtodo implementado sobre sckit-learn asume que la data viene en un conjunto de datos. Sckit-learn incluye algunos conjuntos de datos conocidos.

Datos de entrenamiento, validación y de prueba¶

  • Los datos de entrenamiento(training): son los datos que entrenan los modelos
  • Los datos de validacin (validation) : elecciona el mejor de los modelos entrenados.
  • Los datos de prueba (testing): Nos ofrece el error real cometido con el modelo seleccionado.

Hastie, Tibshirani y Friedman en su libro The Elements of Statistical Learning indican que una divisin tpica para estos conjuntos puede ser de 50% para el entrenamiento y 25% para la validacin y prueba, respectivamente.

API de scikit-learn¶

Los principales "interfaces" en scikit-learn son (una clase puede implementar mltiples interfaces):

  • Estimador: Un estimador es un objeto que se ajuste a un modelo basado en algunos datos de entrenamiento y es capaz de inferir algunas propiedades de los nuevos datos.

    estimador = obj.fit(data, targets)

  • Predictor:

    prediccion = obj.predict(data)

  • Transformador:

    n_data = obj.transform(data)

  • Modelo:

    s = obj.score(data)

Algoritmos en machine learning¶

Aprendizaje supervisado¶

Estos algoritmos consisten en una variable de salida/resultado (o variable dependiente) que ha de ser predicha a partir de un conjunto dado de predictores (variables independientes). Usando este conjunto de variables, generamos una funcin que se asignan las entradas a las salidas deseadas. El proceso de formacin contina hasta que el modelo alcanza un nivel deseado de precisin en los datos de entrenamiento. Algunos ejemplos:

  • Linear Regression.
  • Logistic Regression.
  • KNN.
  • Random Forest.

Aprendizaje no supervisado¶

En estos algoritmos, no tenemos ninguna variable objetivo o resultado para predecir/estimar. Se utiliza para la agrupacin de la poblacin en diferentes grupos, lo que es ampliamente utilizado para la segmentacin de clientes en diferentes grupos por intervencin especfica. Algunos ejemplos :

  • K-Means.
  • Dimensionality Reduction Algorithms.

Aprendizaje por refuerzo¶

Utilizando este algoritmo, la mquina est capacitada para tomar decisiones especficas. Funciona de la siguiente manera: la mquina est expuesta a un ambiente donde se entrena a s misma continuamente mediante ensayo y error. Esta mquina aprende de la experiencia pasada y trata de captar el mejor conocimiento posible para tomar decisiones de negocio precisos. Un ejemplo:

  • Procesos de decisin de Markov.
  • Metodos de Montecarlo.

Validación de modelos¶

Una pieza importante de machine learning es la validacin del modelo: es decir, determinar qu tan bien el modelo puede generalizar a partir de los datos de entrenamiento futuros datos no etiquetados. Veamos un ejemplo usando el clasificador de vecino ms cercano (nearest neighbor classifier). Este es un clasificador muy simple: lo que hace es almacenar todos los datos de entrenamiento, y para cualquier cantidad desconocida, simplemente devuelve la etiqueta del punto de entrenamiento ms cercano.

Regresión lineal¶

Se utiliza para estimar los valores reales (costo de las casas, nmero de llamadas, las ventas totales, etc.) en base a la variables continuas. Aqu se establece una relacin entre las variables dependientes e independientes ajustando una lnea. Esta lnea de mejor ajuste se conoce como lnea de regresin y es representado por una ecuacin lineal:

Y = a*X + b

En esta ecuacin:

  • Y : Variable dependiente
  • a : Pendiente
  • X : Variable independiente
  • b : Intercepto

Estos coeficientes a y b son derivados basados en la minimizacin de la suma de la diferencias de cuadrados de la distancia entre los puntos de los datos y la lnea de regresin.

La regresin lineal es principalmente de dos tipos: regresin lineal simple y regresin lineal mltiple.

La regresin lineal simple se caracteriza por una variable independiente, mientras que la regresin lineal mltiple se caracteriza por mltiples variables independientes, adems para encontrar la mejor lnea de ajuste, podemos fijar una regresin polinmica o curvilnea.

Una funcin de coste, tambin llamada funcin de prdida (loss function), se utiliza para definir y medir el error de un modelo. Las diferencias entre los valores del modelo y los valores observados en el conjunto de entrenamiento se denominan errores de entrenamiento, las diferencias entre los valores predichos y observados en los datos de prueba son llamados errores de prediccin o errores de prueba.
