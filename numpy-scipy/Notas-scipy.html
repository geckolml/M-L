El método del gradiente¶

El mtodo del gradiente es un algoritmo de optimizacin utilizado para encontrar el mnimo local de una funcin, que se utiliza en muchos algoritmos del machine learning.

El gradiente (o Jacobiano) en un punto indica la direccin para la que hay un mayor grado de inclinacin. Dado que estamos buscando un mnimo, una posibilidad es la de tomar un paso en la direccin opuesta al gradiente. Estamos ponderando el tamao del paso por un factor $\alpha$ conocido en machine learning como tasa de aprendizaje.

Si $\alpha$ es pequeo, el algoritmo puede converger hacia un mnimo local, tomando mucho tiempo. Si $\alpha$ es grande, el algoritmo puede converger ms rpido, pero tambin puede sobrepasarse y nunca encontrar el mnimo.

Este mtodo es de primer orden, ya que se requiere el clculo de la primera derivada en cada iteracin.

Como se menciona en algunos textos, algunos algoritmos determinan el valor apropiado de $\alpha$ en cada escenario (paso) realizando una bsqueda lineal:

$$ \alpha^* = \operatorname*{arg\,max}_{\alpha}f(x_k -\alpha \nabla f(x_k)) $$

el cual es un problema de optimizacin 1D.

    """ Version generalizada de la funcion de  Rosenbrock """

# Ejemplo tomado desde people.duke.edu

def rosen_der(x):
    """Derivada de la funcion Rosen generalizada."""
    xm = x[1:-1]
    xm_m1 = x[:-2]
    xm_p1 = x[2:]
    der = np.zeros_like(x)
    der[1:-1] = 200*(xm-xm_m1**2) - 400*(xm_p1 - xm**2)*xm - 2*(1-xm)
    der[0] = -400*x[0]*(x[1]-x[0]**2) - 2*(1-x[0])
    der[-1] = 200*(x[-1]-x[-2]**2)
    return der

from scipy import linalg as la
def mdg(fun, x0, args=(), maxfev=None, alpha=0.0002,maxiter=100000, tol=1e-10, callback=None, **options):
    
    mejorx = x0
    mejory = fun(x0)
    funcllam = 1
    niter = 0
    mejor = True
    alto = False

    while mejor and not alto and niter < maxiter:
        niter += 1
        # las proximas  2 lineas son descenso de gradiente
        paso = alpha * rosen_der(mejorx)
        mejorx = mejorx - paso

        mejory = fun(mejorx)
        funcllam += 1

        if la.norm(paso) < tol:
            mejor= False
        if callback is not None:
            callback(mejorx)
        if maxfev is not None and funcllam>= maxfev:
            alto = True
            break

    return opt.OptimizeResult(fun=mejory, x=mejorx, nit=niter,
                              nfev=funcllam, success=(niter > 1))

Integración de MonteCarlo¶

Supongamos que queremos encontrar los valores de $\int_{a}^{b}f(x)dx$ en alguna regin con volumen V. La integracin de Montecarlo estima esta integral, calculando la fraccin de puntos que caen por debajo de $f(x)$ multiplicado por V.

En el contexto estadstico, la integracin de Montecarlo, se utiliza para la estimacin de la esperanza: Sea $g:\mathbb{R} \rightarrow \mathbb{R}$, por definicin la esperanza es:

$$ E_f(g(X)) = \int_{X}g(x)f_{X}(x)dx $$

con $$ \bar{g_n} = \frac{1}{n}\sum_{i = 1}^{n}g(x_i), $$

donde $x_i \sim f$ de la funcin densidad $f$.

Podemos estimar la varianza de Montecarlo, de la aproximacin como:

$$ v_n = \frac{1}{n^2}\sum_{i =1}^{n}[h(x_i) - \bar{g_n}]^2. $$

De manera ms compacta, debido al teorema del Lmite Central, para $n$ muy grande

$$ \frac{\bar{g_n} - E_{f}(g(X))}{\sqrt{v_n}} $$

es aproximadamente distribuida como un variables $N(0,1)$y esto conduce a la construccin de un test de convergencia y lmites de confianza sobre la aproximacin $E_{f}(g(X))$. La convergencia de la integracin de Monte Carlo es $O(n^{1/2})$ y es independiente de la dimensin. De ah que la integracin Monte Carlo resulta til para la integracin de moderada y alta dimensin desde la integracin numrica converge a $O(n^d)$.

Incluso para los problemas de baja dimensin, la integracin de Monte Carlo puede tener una ventaja cuando el volumen a ser integrado est concentrado en una regin muy pequea y podemos utilizar la informacin de la distribucin para extraer muestras con mayor frecuencia en la regin de importancia.

Veamos un ejemplo: La distribucin de Cauchy es dado por

$$ f(x) =\dfrac{1}{\pi(1 + x^2)},\ \ -\infty < x < \infty $$

y supongamos que queremos calcular $P(X>4)$ usando Monte Carlo, es decir queremos estimar la cantidad:

$$ \int_{4}^{\infty}\dfrac{1}{\pi(1 + x^2)}dx $$

usando a sustitucin $y = 4/x$ se tiene

$$ \int_{0}^{1}\dfrac{4}{\pi(16 + y^2)}dy $$

As que en estimador MC es

$$ \dfrac{1}{n}\sum_{i=1}^{n}\dfrac{4}{\pi(16 + y_{i}^2)} $$

donde $y_i \sim Uniform(0,1)$.

Jerarquía de clusters¶

Clustering es una tcnica de data minning, cuyo proceso consiste en la divisin de datos en grupos de objetos similares. Cuando se representan la informacin obtenida a travs de clusters se pierden algunos detalles de los datos, pero a la vez se simplifica dicha informacin.

Las tcnicas de Clustering son las que utilizando algoritmos matemticos se encargan de agrupar objetos. Usando la informacin que brindan las variables que pertenecen a cada objeto se mide la similitud entre los mismos, y una vez hecho esto se colocan en clases que son muy similares internamente (entre los miembros de la misma clase) y a la vez diferente entre los miembros de las diferentes clases.

Un algoritmo importante se llama k-means:

Este algoritmo debe definir el nmero de clusters que se desean obtener, as se convierte en un algoritmo voraz para particionar. Este algoritmo funciona de la siguiente manera: primeramente se determina la cantidad de clusters en los que se quiere agrupar la informacin, en este caso las simulaciones. Luego se asume de forma aleatoria los centros por cada clusters. Una vez encontrados los primeros centroides el algoritmo har los tres pasos siguientes:

1 . Determina las coordenadas del centroide.

2 . Determina la distancia de cada objeto a los centroides.

3 .Agrupa los objetos basados en la menor distancia.

Es una herramienta para identificar estructuras que estn anidadas dentro de otras grandes estructuras. Hagamos un ejemplo para generar un sistema mltiple de cluster. Para emplear la funcin (hierarchy) que permite esto, hay que construir la MATRIZ DISTANCIA.

# Creando un cluster de una funcion clusters

def clusters(numbers = 20, cnumber = 5, csize  = 10):
    
    rnum  =np.random.rand(cnumber, 2)
    rn =rnum[:, 0] * numbers
    rn = rn.astype(int)
    rn[np.where(rn < 5)] = 5
    rn[np.where(rn > numbers/2.0)] = round(numbers/2., 0)
    ra = rnum[:, 1] * 2.9
    ra[np.where(ra < 1.5)]  =1.5
    
    cls = np.random.rand(numbers, 3) * csize
    
# Multiplicadores aleatorios para puntos centrales del cluster

    rxyz = np.random.rand(cnumber - 1, 3)
    for i in range(cnumber - 1):
        tmp = np.random.rand(rn[i +1], 3)
    
        x = tmp[:, 0] + (rxyz[i, 0] * csize)
        y = tmp[:, 1] + (rxyz[i, 1] * csize)
        z = tmp[:, 2] + (rxyz[i, 2] * csize)
    
        tmp = np.column_stack([x ,y, z])
        cls = np.vstack([cls, tmp])

    return cls

Matrices Sparse¶

Qu pasara si queremos manejar matrices de $10^{10}$ elementos?.

Si la matriz tiene muchos ceros, entonces es posible manejar de manera eficiente este tipo de matriz, que es llamada Sparse.

In [18]:
import numpy as np
from  scipy.sparse.linalg import eigsh
from  scipy.linalg import eigh
import scipy.sparse
import time

N = 300
#  Creamos una matriz aleatoria de tipos sparse

m = scipy.sparse.rand(N,N)
                     
a = m.toarray() # clon de Numpy para m
                      
print("El array de NUmpy tiene"  + str(a.nbytes) + ' bytes')
print("El matriz sparse tiene"  + str(m.data.nbytes) + ' bytes')

# No-sparse

t0 = time.time()
res1 = eigh(a)
dt = str(np.round(time.time() -t0, 3)) + 'segundos '
print("La operaciones no-sparse toman" + dt)

# sparse

t0 = time.time()
res1 = eigsh(m)
dt = str(np.round(time.time() -t0, 3)) + 'segundos '
print("La operaciones sparse toman" + dt)

Scipy contiene en el mdulo scipy.stats una extensa coleccin de herramientas para la estadistica y las probabilidades. como es el caso de las distribuciones continuas, discretas y multivariadas.

Cuando llamamos a distribucin desde scipy.stats, podemos extraer informacin de varias maneras: PDF, CDF, RVS.

$$ PDF = e^{(-x^2/2)/\sqrt{2\pi}} $$
In [7]:
# Ejemplo de probabilidades: PDF de la RV normal

import numpy as np
from scipy.stats import norm

# Rango de la muestra a tomar

x = np.linspace(-5, 5, 100000)

# Colocando los parametros para la distribucion normal

dist = norm( loc = 0, scale = 1)

# Recuperando PDF y CDF de la distribucion normal

pdf = dist.pdf(x)
cdf = dist.cdf(x)

# Recupperando RVS ( Muestreo de la variable aleatoria)
muestreo = dist.rvs(500)
