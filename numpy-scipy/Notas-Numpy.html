Numpy¶

Numpy es el paquete fundamental de Python para computacin cientfica.

Agrega la capacidad de usar arrays N dimensionales, broadcasting (operaciones elemento a elemento), operaciones matemticas usadas en el lgebra Lineal y la conexin con otros lenguajes de programacin como Fortan, C/C++.

Los arrays de Numpy tienen ciertas ventajas sobre las listas y diccionarios en la manera de almacenar datos, debido a que proporciona un objecto de almacenamiento de datos conocido como ndarray.

    return alista

np.darray¶

  • Bloques contiguos de memoria (seccin de memoria) que almacenan una coleccin de datos:

  • El objeto ndarray es similar a las listas, pero slo permite que el mismo tipo de elementos puedan ser almacenados en columnas. Un array es un objeto de Python que es una vista en esta seccin de la memoria

  • Algunos atributos:
    • data
    • shape
    • strides
    • dtype
    • size
    • ndim
  • El objeto ndarray permite diferentes puntos de vista (sub sections, reshaping, transposition, etc.) sin cambiar nada en memoria slo la interpretacin de los cambios de memoria almacenados.
  • Todas las interfaces con esta estructura de datos se escriben en C y el acceso y modificacin de las estructuras de datos son por lo tanto bastante rpidas.

Reshaping y resizing¶

Reshaping(remodelado) es una tcnica para remodelar un array y alterar la manera que son interpretados.

In [105]:
a2 = np.array([[1, 2], [3, 4]])
np.reshape(a2, (1, 4))
Out[105]:
array([[1, 2, 3, 4]])
In [107]:
a2.reshape(4)
Out[107]:
array([1, 2, 3, 4])

Podemos reformular los arrays de diversas maneras. Por ejemplo si tenemos un array de 25 elementos, podemos expresar esto como una matriz 5 x 5 o como una matriz de 3 dimensiones.

Considerar una funcin que simula a N caminantes al azar cada uno tomando M pasos y que alcula la distancia ms alejada del punto de partida obtenido por cualquiera de los caminantes aleatorios:

In [191]:
import numpy as np

def caminante_aleatorio_max_distancia(M,N):
    
    """
    Simula N caminantes, tomando n pasos y que devuelve la
    distancia mas alejada tomada desde el punto de partida
    de un caminante aleatorio
    """
    
    # random.randn devuelve numeros desde una distribucion normal
    camino = [np.random.randn(M).cumsum() for _ in range(N)]
    return np.max(np.abs(camino))
